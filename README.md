### 📌 项目简介

本项目通过 **Telegram 机器人** 将 Web 端与TG端处理流程连接起来，构建一个简单高效的订单处理系统，实现了订单分配自动化，同时避免了终端买家与供应商的直接自由联系

*  **Web 页面**：用户通过网页提交订单
*  **Telegram Bot**：将订单实时推送至 Telegram Bot
*  **人工处理**：供应商工作人员先到先得，抢到订单后在 Telegram 中处理订单
*  **互动反馈**：Web端提供“撤销”、“催单”、“质疑”功能；TG端处理结果（成功 / 失败）返回给用户
*  **管理员功能**：Web端-买家余额管理；TG端-工作人员薪资管理
---

### 🔄 工作流程

```text
[ 用户（Web） ]
        │
        ▼
[ 提交订单 ]
        │
        ▼
[ 后端服务器 ]
        │
        ▼
[ Telegram 机器人 ]
        │
        ▼
[ 工作人员处理 ]
        │
        ▼
[ 处理结果（成功 / 失败） ]
        │
        ▼
[ 返回用户 ]
```
